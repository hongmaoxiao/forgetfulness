# 健忘杀手

> 定时发送短信提醒，以对抗健忘。

## 使用

后端用go，前端用vue。go的代码没上传。

数据库用mysql，注册登录等用了redis。

不支持本地运行，只能在线上使用，地址是[[https://weidayouyi.com](https://weidayouyi.com/)]([https://weidayouyi.com](https://weidayouyi.com/))，目前每个人有20条短信免费量。后期打算做个小程序版的，支持弹出微信提醒，大概像潮汐日历一样，完全免费。
